About m0ssion

You are using m0ssion, a minimalistic extensible in-browser framework for the Galaxy chat.
The aim of m0ssion is to provide an easy and powerful tool for those who wants to improve their experience with Galaxy chat or are annoyed by some «features» like permanent money sucking, silly games, annoying appearing objects or disturbing notifications. Nevertheless, this framework can do almost everything with the chat. It depends only on your fantasy and programming skills.

Documentation

m0ssion is easy as hell. Really. If you want to learn how to write modules for it, visit our Wiki on GitHub. Basic examples and description of Galaxy's JS are provided as well.
